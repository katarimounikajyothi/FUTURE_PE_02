# FUTURE_PE_02
AI-Powered Beauty Salon Content Pack

Project Overview

This project was created as part of the Prompt Engineering Internship by Future Interns.

The goal of this project is to use AI tools like ChatGPT to create engaging website and promotional content for a local beauty salon.

---

Business Chosen

GlowUp Beauty Salon — a modern beauty salon in Warangal offering skincare, haircare, bridal makeup, and self-care services.

---

Features

- Website homepage copy
- Beauty salon service descriptions
- CTA-focused content
- AI-generated captions
- Reusable prompts
- Social media promotional content
- Marketing hooks

---

Tools Used

- ChatGPT
- GitHub
- Lovable AI

---

Project Structure

outputs/ → generated content
prompts/ → prompts used
hooks.txt → beauty marketing hooks
captions.txt → social media captions
README.md → project overview

---

Learning Outcomes

- Improved prompt engineering skills
- Learned AI-based content generation
- Understood beauty brand marketing workflows
- Practiced creating promotional website content
- Learned GitHub project organization

---

Platforms Covered

- Instagram
- Facebook
- Website Landing Pages
- Promotional Marketing Content

---

Outcome

This project helped in understanding how prompt engineering and AI tools can be used to create professional beauty salon website content and social media marketing material for local businesses like GlowUp Beauty Salon.
